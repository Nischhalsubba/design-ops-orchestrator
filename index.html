02. Workflow Commands

npm start
Dev Mode. Launches local server (localhost:3000), watches for changes, and activates AI Healer.
npm run build
Production Build. Minifies CSS/JS/HTML, generates sitemaps, optimizes images (AVIF/WebP), and creates reports.
npm run lint
Audit Code. Runs ESLint, Stylelint, and Pug-Linter to ensure strict code standards.
npm test
Run Tests. Executes Jest unit tests and BackstopJS visual regression tests.
