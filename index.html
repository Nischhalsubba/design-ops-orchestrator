03. Quick Setup

1

Clone the Repo

Open your Terminal (Command Prompt) and paste this:

git clone https://github.com/Nischhalsubba/design-ops-orchestrator.git
2

Install Dependencies

This downloads Gulp, Sass, and all the image optimization tools.

npm install
3

Connect AI (Optional)

To let the AI fix your code errors, create a .env file and add your key:

API_KEY=your_gemini_key

04. Commands

npm start

For Creation

Run this while you are designing/coding. It opens a browser window that updates instantly when you save files.

npm run build

For Handoff

Run this when you are done. It optimizes everything, creates critical CSS, and prepares the `dist/` folder for your engineers.
