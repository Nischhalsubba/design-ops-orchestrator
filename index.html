01. Setup Guide

1

Clone the Repository

Start by getting the code on your local machine.

git clone https://github.com/Nischhalsubba/design-ops-orchestrator.git
cd design-ops-orchestrator
2

Install Dependencies

This project uses npm to manage over 200 development tools including Gulp, Sass, and Gemini SDK.

npm install
3

Configure AI Environment

The "Healer" and "Architect" bots require a Google Gemini API Key.
1. Create a file named .env in the root.
2. Add your key: 

API_KEY=your_gemini_api_key_here

02. Workflow & Commands

npm start

Runs the Development Server.

  • Compiles Pug, SCSS, TS in memory
  • Starts Localhost:3000 (BrowserSync)
  • Watches files for changes
  • Activates AI Healer for build errors

npm run build

Runs the Production Build.

  • Minifies & Obfuscates Code
  • Generates Source Maps
  • Optimizes Images (WebP/AVIF)
  • Injects Critical CSS

npm run lint

Runs the Quality Assurance.

  • Stylelint (SCSS Standards)
  • ESLint (JavaScript Logic)
  • Pug-Linter (Template Syntax)

npm test

Runs Testing Suites.

  • Jest (Unit Tests)
  • BackstopJS (Visual Regression)

05. AI Integration (Gemini 3)

üíä The Healer Bot

Located in gulp/utils/ai-healer.js. This module wraps your build tasks. If a task like Sass compilation fails, it:

  1. Captures the Error Stream.
  2. Sends the error + stack trace to Google Gemini 3 Flash.
  3. Receives a corrected code snippet.
  4. Prints the solution directly in your terminal.
[AI HEALER]: You missed a semicolon in main.scss line 42.
Fix: $color: #fff;

Powered by js/ai-manager.js. This is the Chat Widget you see in the bottom right corner.

  • Use case: Ask it to generate a new Gulp task or explain a specific SCSS mixin.
